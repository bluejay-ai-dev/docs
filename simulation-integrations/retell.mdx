This guide walks you through running Bluejay simulations directly against your Retell voice agents. Once configured, every simulation Bluejay runs against a Retell-bound agent will dial outbound through Retell automatically, and every tool call the agent makes during the conversation will land in Bluejay's Tool Adherence tab.

### Quick Prerequisites

- **Retell API Key with the Webhook badge**
- **At least one Retell agent with a phone number bound to it**
- **A Bluejay Account**

### Setting Up the Integration

### 1. Generate Your Retell API Key

1. In Retell, go to **Settings** then **API Keys**, and generate a new API key.
2. Ensure the key has the **Webhook** badge next to it (Bluejay uses it to verify webhook signatures).

<video autoPlay muted loop playsInline className="w-full rounded-xl" src="/videos/retell-sim-1.mp4"></video>

### 2. Bind a Phone Number to Your Agent in Retell

1. In Retell, go to **Phone Numbers** and click into your phone number.
2. Set both **Inbound Call Agent** AND **Outbound Call Agent** to your agent.
3. Save.

<video autoPlay muted loop playsInline className="w-full rounded-xl" src="/videos/retell-sim-2.mp4"></video>

### 3. Set the Webhook URL on Your Retell Agent

1. In Retell, open your agent and click **Webhook Settings**.
2. Paste this URL into **Agent Level Webhook URL**:

```
https://api.getbluejay.ai/v1/integrations/retell
```

3. Keep the default events enabled: `call_started`, `call_ended`, `call_analyzed`.
4. Save and publish the agent.

<video autoPlay muted loop playsInline className="w-full rounded-xl" src="/videos/retell-sim-3.mp4"></video>

### 4. Save Your Key in Bluejay and Sync

1. In Bluejay, go to **Settings**, then **Integrations**, then the **Retell** tab.
2. Paste your Retell API key and click **Save Changes**.
3. Click **Sync Agents from Retell** and confirm in the prompt.
4. Within 5 to 15 seconds, every Retell agent in your workspace appears in Bluejay as a native agent.

<video autoPlay muted loop playsInline className="w-full rounded-xl" src="/videos/retell-sim-4.mp4"></video>

#### Quick Sync from the Plus Menu

For a faster path, you can sync your Retell agents directly from the quick add menu inside Bluejay.

1. Click the **+** button in the top left corner.
2. Select **Retell**.
3. Paste your Retell API key.
4. Click **Save and Sync** to import your Retell agents into Bluejay.

<video autoPlay muted loop playsInline className="w-full rounded-xl" src="/videos/retell-sim-5.mp4"></video>

### 5. Create and Run Your Simulation

With your Retell agent synced into Bluejay, you can run a simulation against it.

1. From your agent in Bluejay, click **Create Simulation**. The create flow opens with several configuration tabs across the top.
2. Open the **Goal Adherence Rate** tab. Goal Adherence Rate reads the prompt from your agent so it can auto-generate a digital human persona aligned with the conversation goals.
3. Click **Auto-generate**.
4. The persona count defaults to **5**. Choose **1** or any number you prefer.
5. Click **Save**.
6. Click **Generate Digital Human**. After a moment, the new digital human appears in the panel.
7. Confirm the digital human is selected, then click **Run** to dispatch the call.
8. Wait for the simulation to complete. When it finishes, the result opens with the transcript, tool calls, and metric scores. Review it the same way you would any other simulation in Bluejay.

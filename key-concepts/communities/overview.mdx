Communities are reusable collections of Digital Humans grouped around a shared testing purpose. They make it easier to run the same benchmark set across multiple agents and evaluation cycles.

## how it is used

You create Communities to organize personas by audience, language, or scenario type, then reuse them in simulations and comparisons so your testing stays consistent over time.

This guide will walk you through how to connect your Retell voice agents to Bluejay's observability software. You can select Retell as your Voice AI Provider in the Advanced settings tab of an existing agent.

<video controls className="w-full aspect-video rounded-xl" src="/videos/retell-integration.mp4"></video>


### Quick Prerequisites

- **Retell API Key**
- **A Bluejay Account**

### Setting Up the Integration

### 1. Configure Organization Settings

1. Log into your **Bluejay Dashboard**
2. Click on your **Organization** button in the bottom left corner
3. Select the **Integrations** option 


![Navigate to Integrations](/images/navigate-integrations.png)

### 2. Adding Your Retell Key

1. Locate the **Retell API Key** input
2. Enter your Retell API Key into the field 
3. Click **Save API Key**

![Save API Key](/images/save-api-key.png)

### 3. Configuring Your Agent in Retell

1. **Copy the Webhook URL given** in the Webhook configuration dropdown 
2. Navigate to your Retell Agents dashboard and click the **Webhook Settings** dropdown
3. **Paste the Webhook URL into the Agent Level Webhook URL field**
<video controls className="w-full aspect-video rounded-xl" src="/videos/retell-webhook.mov"></video>

### 4. Configuring Your Agent in Bluejay

1. Navigate to your desired Agents **Settings** tab
2. Scroll down and click the **Advanced Settings** dropdown
3. Select **Retell as your Voice AI Provider** 
4. Copy and paste your Retell Agents ID into the **Assistant ID** field
5. Click **Save Changes**

<video controls className="w-full aspect-video rounded-xl" src="/videos/set-voice-provider.mov"></video>

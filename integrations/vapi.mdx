This guide will walk you through how to connect your Vapi voice agents to Bluejay's observability software. You can select Vapi as your Voice AI Provider when in the Advanced settings tab of an existing agent.

{/* <video controls className="w-full aspect-video rounded-xl" src="/videos/retell-integration.mp4"></video> */}


### Quick Prerequisites

- **Vapi API Key**
- **A Bluejay Account**

### Setting Up the Integration

### 1. Configure Organization Settings

1. Log into your **Bluejay Dashboard**
2. Click on your **Organization** button in the bottom left corner
3. Select the **Integrations** option 


![Navigate to Integrations](/images/integrations.png)

### 2. Adding Your Vapi Key

1. Locate the **Vapi API Key** input
2. Enter your Vapi API Key into the field 
3. Click **Save API Key**

![Save API Key](/images/enter-vapi.png)

### 3. Configuring Your Agent in Vapi

1. **Copy the Webhook URL given** in the Webhook configuration dropdown 
2. Navigate to your Vapi Agents dashboard and scroll down to the **Messaging** dropdown
3. **Paste the Webhook URL into the Server URL field**
4. Click **Add Header** and enter **secret** as the header name and your Vapi public API key as the Value

<video controls className="w-full aspect-video rounded-xl" src="/videos/configure-vapi-agent.mov"></video>

### 4. Configuring Your Agent in Bluejay

1. Navigate to your desired Agents **Settings** tab
2. Scroll down and click the **Advanced Settings** dropdown
3. Select **Vapi as your Voice AI Provider** 
4. Copy and paste your Vapi Agents ID into the **Assistant ID** field
5. Click **Save Changes**

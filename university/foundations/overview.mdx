## What you'll learn

By the end of this lesson you will be able to:

- Navigate the Bluejay dashboard and understand the workspace layout
- Generate an API key and create your first Agent
- Understand how Agents, Digital Humans, Simulations, Communities, and Custom Metrics relate to each other
- Identify where each entity lives in the dashboard

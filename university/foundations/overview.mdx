## What you'll learn

By the end of this course you will be able to:

- Navigate the Bluejay dashboard and understand the workspace layout
- Create and configure an Agent
- Understand how Agents, Digital Humans, Simulations, and Custom Metrics relate to each other
- Set up the building blocks needed for testing and monitoring

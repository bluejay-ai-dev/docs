## Objectives

- Log in and explore the Bluejay dashboard
- Locate the API key page and generate a key
- Create your first Agent

## Walkthrough

<Steps>
  <Step title="Log in to Bluejay">
    Navigate to [app.getbluejay.ai](https://app.getbluejay.ai) and sign in with the credentials provided during onboarding. You'll land on the **Dashboard** — this is your home base for simulations, observability, and metrics.
  </Step>
  <Step title="Generate an API key">
    Open **Settings → API Keys** and create a new key. Copy it somewhere safe — you'll use it to authenticate API calls and integrations later.
  </Step>
  <Step title="Create your first Agent">
    Go to **Agents** and click **New Agent**. Give it a name and description that reflects the voice or chat agent you want to test. An Agent in Bluejay is a mirror of your production agent — it holds the configuration that simulations and observability evaluate against.

    For a deeper look at what Agents represent, see [Key Concepts → Agents](/key-concepts/agents/overview).
  </Step>
</Steps>

## Knowledge check

<AccordionGroup>
  <Accordion title="What is the purpose of an API key in Bluejay?">
    The API key authenticates requests from your code, CI pipelines, and integrations against your Bluejay workspace. Keep it secret and rotate it if compromised.
  </Accordion>
  <Accordion title="What does an Agent represent?">
    An Agent is Bluejay's representation of a conversational AI you want to test or monitor. It holds configuration like connection type, evaluation metrics, and associated simulations.
  </Accordion>
</AccordionGroup>

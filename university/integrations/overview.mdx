## What you'll learn

By the end of this course you will be able to:

- Connect your agent to Bluejay via telephony, SIP, LiveKit, websockets, or HTTP
- Set up provider-specific observability integrations (Retell, Vapi, Bland, ElevenLabs)
- Route alerts and reports to Slack

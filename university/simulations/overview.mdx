## What you'll learn

By the end of this lesson you will be able to:

- Create a Simulation, attach Digital Humans, and run it against an Agent
- Review transcripts and metric scores from a completed simulation run
- Choose the right simulation type for your testing scenario
- Build a test plan that covers happy paths, edge cases, and failure modes
- Organize Digital Humans into Communities for reuse across simulations

## What you'll learn

By the end of this lesson you will be able to:

- Connect production calls to Bluejay for automatic evaluation
- Create Custom Metrics tailored to your business needs
- Use Metrics Lab to prototype and refine metrics before promoting them
- Build dashboards that visualize agent quality trends
- Set up alerts and Slack notifications to catch quality regressions in real time

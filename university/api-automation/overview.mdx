## What you'll learn

By the end of this lesson you will be able to:

- Authenticate with the Bluejay REST API and make common requests
- Create Agents, queue simulation runs, and retrieve results programmatically
- Set up webhooks to receive real-time notifications
- Build Workflows that automate recurring tests on a schedule
- Connect voice provider and simulation transport integrations

## What you'll learn

By the end of this course you will be able to:

- Use the Bluejay REST API to manage Agents, Simulations, and Metrics programmatically
- Trigger simulation runs from CI/CD pipelines
- Set up Workflows and Schedules for automated, recurring evaluation

## Welcome

Bluejay offers two APIs:
- [Mimic: Simulation API](https://bluejay.mintlify.app/api-reference/endpoint/queue-simulation)
- [Skywatch: Observability API](https://bluejay.mintlify.app/api-reference/endpoint/evlauate-call)

## Authentication

Sign up for a Bluejay account at [app.getbluejay.ai](https://app.getbluejay.ai) and 
generate an API key from the [API Keys page](https://app.getbluejay.ai/api-keys).

---
title: 'Update workflow'
description: 'Partially update a workflow; replacing definition re-validates the same typed React Flow graph as create.'
openapi: 'PUT /v1/workflow/{workflow_id}'
---

Omitted fields are left unchanged. If `agent_ids` appears in the body (including `[]`), all agent links are **replaced**; `[]` clears links. If `definition` appears, it is **fully replaced** and validated like **POST** `/v1/workflow` (same discriminated node and `data` types as [Create workflow](/api-reference/endpoint/create-workflow-v2)).

Sends only the fields you want to change; include `definition` only when replacing the entire React Flow graph.

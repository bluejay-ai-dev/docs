## Move Agent to Folder — PUT /v1/folders/move-agent

> **What this endpoint does:** Move an agent to a folder or disassociate it from any folder. - If folder_id is provided: Associates the agent with that folder - If folder_id is null: Disassociates the agent from any folder Both the agent and folder (if provided) must belong to the user's organization.

This endpoint lists **legacy** automation-style workflows from **GET** `/v1/workflows`. For conversation graphs (React Flow definitions), use [List workflows](/api-reference/endpoint/list-workflows-v2) (**GET** `/v1/workflow`) instead.

Returns workflows for your organization, optionally filtered by `agent_id`. See the OpenAPI schema on this page for the response shape.

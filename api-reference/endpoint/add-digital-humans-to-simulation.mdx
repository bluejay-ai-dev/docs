## Add Digital Humans to Simulation — POST /v1/simulation/{simulation_id}/add-digital-humans

> **What this endpoint does:** add existing digital humans to a simulation without replacing existing associations.

**Endpoint:** POST `https://api.getbluejay.ai/v1/simulation/{simulation_id}/add-digital-humans`
**Auth:** `X-API-Key` header
**Content-Type:** application/json

### Required Parameters
| Name | Type | Description |

| digital_human_ids | array[integer] | array of existing digital human IDs to add to the simulation |

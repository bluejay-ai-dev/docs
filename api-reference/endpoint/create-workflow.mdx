## Endpoint Details
- **Method:** POST
- **URL:** `https://api.getbluejay.ai/v1/workflows`
- **Auth:** API Key via `X-API-Key` header
- **Content-Type:** application/json

## Parameters
| Name | Location | Type | Required | Description |
|------|----------|------|----------|-------------|
| X-API-Key | header | string | yes | API key required to authenticate requests. |
| name | body | string | yes | Name of the workflow |
| workflow | body | object | yes | Input model for workflow data (with optional position fields) |
| agent_ids | body | array[integer] (nullable) | no | List of agent IDs this workflow belongs to (can be empty or null for no associations) |
| enabled_for_evaluation | body | boolean (nullable) | no | Whether this workflow should be used for agent evaluation (default: True) |

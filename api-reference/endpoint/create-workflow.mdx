This endpoint is **deprecated**. For new integrations, use [Create workflow](/api-reference/endpoint/create-workflow-v2) (`POST /v1/workflow`) with the React Flow graph format described there and in the [Workflows cookbook](/cookbook/workflows).

This legacy endpoint creates an automation-style workflow for an agent (name, `workflow` object with nodes and edges, optional `agent_ids` and `enabled_for_evaluation`). See the OpenAPI schema on this page for the exact request shape.

- **Method:** POST
- **URL:** `https://api.getbluejay.ai/v1/create-digital-human`

| Name | Location | Type | Required | Description |

| X-API-Key | header | string | yes | API key required to authenticate requests. |
| simulation_ids | body | array[integer] (nullable) | no | List of simulation IDs to associate with this digital human |
| simulation_id | body | integer (nullable) | no | ID of the associated simulation. Use simulation_ids instead. |
| digital_human | body | object | yes | Digital human data model for requests (without id and created_at). |

# Integrate Generate Digital Humans — POST /v1/generate-digital-humans

**You are a senior backend engineer. Your task: integrate the endpoint below into the existing codebase with minimal, surgical changes. Do not refactor, rename, or restructure anything beyond what is required for this integration.**

## Endpoint Details
- **Method:** POST
- **URL:** `https://api.getbluejay.ai/v1/generate-digital-humans`
- **Auth:** API Key via `X-API-Key` header
- **Content-Type:** application/json

## Parameters
| Name | Location | Type | Required | Description |
|------|----------|------|----------|-------------|
| X-API-Key | header | string | yes | API key required to authenticate requests. |
| agent_id | body | string | yes | ID of the agent to be used to generate the digital humans |
| simulation_id | body | string (nullable) | no | Optionally attach the digital humans to a simulation |
| prompt_id | body | string (nullable) | no | Prompt UUID associated with the simulation |
| knowledge_base_id | body | string (nullable) | no | Knowledge base UUID associated with the simulation |
| goal_adherence | body | array[object] (nullable) | no | List of goal adherence scenarios |
| workflow_adherence | body | object (nullable) | no | Dictionary of workflow IDs and their counts |
| replay_transcript | body | array[object | object] (nullable) | no | List of transcript replays to generate digital humans from |
| customer_personas | body | array[object] (nullable) | no | List of customer personas to be used in the simulation |
| load_testing | body | integer (nullable) | no | Number of load testing calls (default: 0) |
| red_teaming | body | integer (nullable) | no | Number of red teaming calls (default: 0) |
| traits | body | array[object] (nullable) | no | List of traits to apply to all generated digital humans |
| num_runs | body | integer (nullable) | no | Number of runs per digital human per simulation run (run count). (default: 1) |

````
## Create workflow (v2) — POST /v1/workflows-v2

> **What this endpoint does:** Create a workflows_v2 row. You must send `name` and `definition`; `definition` can be `{}` and may omit `nodes` / `edges` (they become `[]` before persist).

**Endpoint:** POST `https://api.getbluejay.ai/v1/workflows-v2`
**Auth:** `X-API-Key` header
**Content-Type:** application/json

### Body fields
| Name | Required? | Notes |
|------|-----------|--------|
| name | Yes | Non-empty string, max 255 characters |
| definition | Yes | Must be present; may be `{}`. `nodes` and `edges` are optional in JSON—omitted keys normalize to `[]`. |
| description | No | Omit or null |
| agent_ids | No | Omit or null for org-only; if set, agents must be in your org |

Minimal body: `{"name": "My workflow", "definition": {}}`.

Review the full schema at https://docs.getbluejay.ai/api-reference/endpoint/create-workflow-v2. A **400** response may include a `detail` message.

### Example
```python
import requests

    response = requests.post(url, headers=headers, json=payload)
    response.raise_for_status()
    return response.json()
````

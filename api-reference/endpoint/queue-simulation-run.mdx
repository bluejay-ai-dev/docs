## Endpoint Details
- **Method:** POST
- **URL:** `https://api.getbluejay.ai/v1/queue-simulation-run`
- **Auth:** API Key via `X-API-Key` header
- **Content-Type:** application/json

## Parameters
| Name | Location | Type | Required | Description |
|------|----------|------|----------|-------------|
| X-API-Key | header | string | yes | API key required to authenticate requests. |
| simulation_id | body | string | yes | ID of the simulation to be run |
| prompt_id | body | string (nullable) | no | Prompt UUID associated with this run |
| knowledge_base_id | body | string (nullable) | no | Knowledge base UUID associated with this run |
| digital_human_ids | body | array[string] (nullable) | no | Digital human IDs to be used in the simulation |

| sip_uri | body | string (nullable) | no | Optional SIP URI to be used in this simulation run (overrides the agent SIP URI) |
| livekit_agent_name | body | string (nullable) | no | Optional agent name to be used by user's livekit, an empty string will use the default agent |
| livekit_metadata | body | object (nullable) | no | metadata to be passed to the livekit room |
| pipecat_agent_name | body | string (nullable) | no | Pipecat agent name |

| pipecat_room_properties | body | object (nullable) | no | Pipecat room properties |

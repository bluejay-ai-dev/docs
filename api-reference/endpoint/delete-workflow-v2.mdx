## Delete workflow (v2) — DELETE /v1/workflows-v2/{workflow_id}

> **What this endpoint does:** Delete a workflows v2 row if it exists and belongs to your organization.

**Endpoint:** DELETE `https://api.getbluejay.ai/v1/workflows-v2/{workflow_id}`
**Auth:** `X-API-Key` header

This removes the persisted graph whose structure is documented under Workflows v2 components (`WorkflowV2DefinitionGraph` and related discriminated types).

### Example
```python
import requests

def delete_workflow_v2(workflow_id: str, api_key: str) -> dict:
    url = f"https://api.getbluejay.ai/v1/workflows-v2/{workflow_id}"
    headers = {"X-API-Key": api_key}
    response = requests.delete(url, headers=headers)
    response.raise_for_status()
    return response.json()
```

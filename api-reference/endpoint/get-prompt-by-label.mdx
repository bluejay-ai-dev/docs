---
title: 'Get Prompt By Label'
description: ''
openapi: 'GET /v1/agents/{agent_id}/prompts'
---

Retrieves a single prompt by label under `/agents/{agent_id}/prompts?label=...`.

- if label is missing -> 400
- if label resolves to 0 -> 404; >1 -> 400 ambiguous (except latest which should be unique)

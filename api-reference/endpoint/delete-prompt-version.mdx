## Delete Prompt Version — DELETE /v1/agents/{agent_id}/prompts/versions/{version}

> **What this endpoint does:** delete a specific prompt version; cannot delete production; cannot delete only version. if deleting latest, reassign latest to highest remaining version. returns metadata about the deletion.

**Endpoint:** DELETE `https://api.getbluejay.ai/v1/agents/{agent_id}/prompts/versions/{version}`
**Auth:** `X-API-Key` header

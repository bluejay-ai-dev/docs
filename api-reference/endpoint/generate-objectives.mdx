## Generate objectives — POST /v1/generate-objectives

> **What this endpoint does:** Generate objectives for a test case from its intent. **400** if the test case tags indicate workflow playback (tag lowercased contains `workflow`): workflow-tagged digital humans use enriched playback on the agent; objectives are not applicable.

**Endpoint:** POST `https://api.getbluejay.ai/v1/generate-objectives`
**Auth:** `X-API-Key` header
**Content-Type:** application/json

### Required Parameters
| Name | Type | Description |
|------|------|-------------|
| X-API-Key | string | API key required to authenticate requests. |
| test_case_id | integer | ID of the test case |

### Request Body (required fields)
```json
{
  "test_case_id": 123
}
```

Refer to the full schema at https://docs.getbluejay.ai/api-reference/endpoint/generate-objectives. This request body only requires `test_case_id`.

### Example
**POST with body:**
```python
import requests

def generate_objectives(payload: dict, api_key: str) -> dict:
    url = "https://api.getbluejay.ai/v1/generate-objectives"
    headers = {"X-API-Key": api_key}
    response = requests.post(url, headers=headers, json=payload)
    response.raise_for_status()
    return response.json()
```

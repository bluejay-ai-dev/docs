## Endpoint Details
- **Method:** POST
- **URL:** `https://api.getbluejay.ai/v1/create-simulation`
- **Auth:** API Key via `X-API-Key` header
- **Content-Type:** application/json

## Parameters
| Name | Location | Type | Required | Description |
|------|----------|------|----------|-------------|
| X-API-Key | header | string | yes | API key required to authenticate requests. |
| agent_id | body | string | yes | ID of the agent to be used in the simulation |
| name | body | string (nullable) | no | Name of the simulation |
| description | body | string (nullable) | no | Description of the simulation |
| prompt_id | body | string (nullable) | no | Optional default prompt UUID for this simulation |
| knowledge_base_id | body | string (nullable) | no | Optional default knowledge base UUID for this simulation |
| sequential_calling | body | boolean (nullable) | no | Whether to run the simulation with sequential calling (default: False) |
| max_concurrent | body | integer (nullable) | no | Maximum number of concurrent calls in the simulation |
| hangup_on_transfer | body | boolean (nullable) | no | Whether to hang up the call on transfer (default: False) |
| max_call_duration | body | integer (nullable) | no | Maximum duration of each call (default is 10). Units specified by max_call_duration_units (default: 10) |
| max_call_duration_units | body | string (enum: minutes, seconds) (nullable) | no | Units for max_call_duration: 'minutes' (default) or 'seconds' (default: minutes) |
| ringing_timeout | body | integer (nullable) | no | Ringing timeout in seconds for SIP trunks (default 300s) (default: 300) |
| digital_humans_settings | body | object (nullable) | no | Settings for digital humans in the simulation |
| selected_custom_metrics | body | array[string] (nullable) | no | Custom metric IDs to enable for this simulation |
| custom_metrics_failing_threshold | body | integer (nullable) | no | Score threshold below which a custom metric is considered failing (0-100) |
| custom_metrics_passing_threshold | body | integer (nullable) | no | Score threshold at or above which a custom metric is considered passing (0-100) |

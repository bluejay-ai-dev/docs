---
title: 'Get Knowledge Base By Label'
description: ''
openapi: 'GET /v1/agents/{agent_id}/knowledge-bases'
---

Retrieves a single knowledge base by label: `/agents/{agent_id}/knowledge-bases?label=...`.

- if label is missing -> 400
- if label resolves to 0 -> 404; >1 -> 400 ambiguous (except latest which should be unique)

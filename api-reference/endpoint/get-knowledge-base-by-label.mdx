## Get Knowledge Base By Label — GET /v1/agents/{agent_id}/knowledge-bases

> **What this endpoint does:** retrieve a single knowledge base by label: /agents/{agent_id}/knowledge-bases?label=... . - if label is missing -> 400 - if label resolves to 0 -> 404; >1 -> 400 ambiguous (except latest which should be unique)

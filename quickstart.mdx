Bluejay helps teams test, monitor, and improve conversational AI before issues reach production. This quickstart gets you from account creation to your first simulation and production evaluation flow.

## What you'll do

- Create a Bluejay workspace
- Generate an API key
- Run a simulation workflow
- Review observability data from production conversations

## 1. Create your workspace

1. Sign up at [app.getbluejay.ai](https://app.getbluejay.ai).
2. Generate an API key from the [API Keys page](https://app.getbluejay.ai/api-keys).
3. Decide whether you want to start with a simulation workflow, an observability workflow, or both.

<Note>
  If you're onboarding a team, create a shared workspace and standardize naming for
  agents, simulations, and environments before you run your first evals.
</Note>

## 3. Launch your first evaluation flow

1. Create or connect the agent you want to evaluate.
2. Build a simulation or observability pipeline using the API reference.
3. Review results, traces, and metrics to identify failure patterns before they become customer issues.
